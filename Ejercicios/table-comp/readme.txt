Ejercicio del taller
Obtiene datos de una API y los muestra en una tabla
Hace filtrado por nombre (name)
Instrucciones:
  1.- Inicializar el proyecto: npm init
  2.- Instala LitElement: npm i lit-element
  3.- Iniciar el servidor: polymer serve
  4.- Abrir el navegado en la ruta que indica el comando anterior.
